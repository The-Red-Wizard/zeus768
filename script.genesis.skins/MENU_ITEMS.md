### Theme Folders:
- `classic/` - Traditional clean style (Gray)
- `futuristic/` - Modern sci-fi style (Cyan)
- `anime/` - Japanese animation style (Pink)
- `water/` - Blue aquatic theme (Blue)

**To add a new theme:** Create a folder in `/resources/media/` and add your custom icons!
