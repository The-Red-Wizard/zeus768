SALTS - Stream All The Sources
=============================
Revived and Modernized by zeus768 for Kodi 21+

CHANGELOG
=========

v2.0.0 (2026)
-------------
Complete rewrite for modern Kodi:

NEW FEATURES:
- Full Python 3 compatibility (Kodi 21 Omega and above)

TORRENT SCRAPERS (9 sites):
  * 1337x - General torrent site with automatic mirror detection
  * YTS/YIFY - High quality movie torrents (API-based)
  * EZTV - TV show torrents (API-based)
  * TorrentGalaxy - General torrents with mirror support
  * ThePirateBay - Classic torrent site (API + HTML fallback)
  * LimeTorrents - General torrents
  * Torrentz2 - Meta-search engine
  * RARBG - Mirrors/clones of the original site
  * Nyaa - Anime torrents

STREAMING SITE SCRAPERS (Modernized Legacy - 4 sites):
  * PrimeWire - General streaming (mirror auto-detection)
  * WatchSeries - TV show streaming
  * Movie4K/KinoX - General streaming
  * SolarMovie - General streaming

INDEXER AGGREGATORS:
  * Jackett integration - Connect to 400+ torrent indexers
  * Prowlarr integration - Alternative indexer aggregator

DEBRID SERVICE INTEGRATION:
  * Real-Debrid - Full OAuth device flow authentication
  * Premiumize - API key authentication
  * AllDebrid - PIN-based authentication
  * Cached torrent detection for instant playback

TRAKT.TV INTEGRATION (API v2):
  * Device OAuth authentication
  * Watchlist sync (Movies & TV Shows)
  * Collection sync
  * Trending/Popular/Recommended content
  * Custom lists support
  * Calendar (upcoming episodes)
  * Scrobbling support
  * Progress tracking

URL RESOLVER:
  * Now uses ResolveURL (script.module.resolveurl) instead of deprecated URLResolver
  * Full compatibility with modern Kodi

IMPROVED FEATURES:
  * Automatic mirror detection for torrent/streaming sites
  * Quality parsing and sorting (4K, 1080p, 720p, etc.)
  * Size display for all sources
  * Seed/peer count display
  * Cached torrent detection for debrid services

REMOVED:
  * Old dead streaming site scrapers that no longer exist
  * Python 2 specific code (print statements, urllib2, etc.)
  * Deprecated Kodi APIs
  * urlresolver dependency (replaced with resolveurl)

TECHNICAL CHANGES:
  * Updated to xbmc.python version 3.0.0
  * Replaced script.module.urlresolver with script.module.resolveurl
  * Modern requests library for HTTP
  * BeautifulSoup4 for HTML parsing
  * JSON-based settings (Kodi 19+ format)
  * Updated to Trakt API v2

CREDITS:
  * Original addon by tknorris
  * Revived and modernized by zeus768
