v1.3.0 (2026-04-XX) - SKIN STYLES + PICTURE-IN-PICTURE
=======================================================
NEW - FULL-SCREEN EPG WINDOW:
- New Kodi WindowXMLDialog shows a Sky TV / Virgin TV style
  guide with a channel column on the left, programme grid on
  the right and a Picture-in-Picture preview in the top-right.
- Channel navigation (up/down) debounces PiP so the portal is
  not hammered when scrolling fast.
- OK on any item plays the channel full-screen.

NEW - SKIN CHOICE (Guide Settings):
- Skin Style dropdown lets the user pick:
    * Classic     - original directory view
    * Sky TV      - deep Sky blue + light-blue highlight bar
    * Virgin TV   - near-black + Virgin red highlight bar
- Picture-in-Picture toggle (default on) with a sub-toggle
  for "Auto-play preview on channel change" (default on).
- Grid hours slider (1-6) for future grid-view expansion.

IMPLEMENTATION:
- Single script-poseidonguide-main.xml drives both themes;
  colours are injected via Skin.String()s at window onInit.
- resources/lib/guide_window.py contains the GuideSkinWindow
  class plus theme palettes.
- default.py:
    * _resolve_stream_url(bridge, stream_id) - mode-aware
      live URL resolver used for the PiP player.
    * _collect_channels_for_window() - gathers channels + EPG
      with a cancelable progress dialog before showing window.
    * launch_skin_window() called at main() entry; drops
      through to the directory view when theme=classic.


v1.2.0 (2026-04-XX) - STB MAC / Stalker Portal Support
=======================================================
NEW:
- Poseidon Guide now reads the Player's portal_mode setting
  and transparently fetches channels + EPG from either an
  Xtreme Codes server or an STB MAC / Stalker portal.
- PoseidonBridge.get_creds() returns a mode-tagged dict so
  the rest of the guide doesn't need to branch.

INTERNALS:
- In-process MAC adapter handles handshake, channel list
  pagination, genres list and get_short_epg.
- api_call() translates the three actions the guide uses
  (get_live_categories / get_live_streams / get_short_epg)
  to the equivalent Stalker endpoints, returning results in
  the same shape as the existing Xtream responses.
- All logic backwards-compatible: existing Xtreme Codes
  installs continue to work exactly as before.
