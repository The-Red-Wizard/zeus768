# Genesis Kodi Addon

**Author:** zeus768  
**Version:** 2026.04.04.01

## Description
Genesis is a powerful Kodi addon for streaming Movies and TV Shows with full torrent support and debrid service integration.

## Features
- **Torrent-based Scrapers**: 5 reliable torrent sources
  - 1337x (Movies & TV)
  - TorrentGalaxy (Movies & TV)
  - YTS (Movies)
  - EZTV (TV Shows)
  - ThePirateBay (Movies & TV)

- **Debrid Services with PIN Authentication**:
  - Real-Debrid
  - Premiumize
  - AllDebrid
  - TorBox

- **Trakt Integration**:
  - Watchlist sync
  - History tracking
  - Scrobbling
  - Custom lists

- **Cached Torrent Support**: Instant streaming from debrid cache

## Installation
1. Download the addon zip file
2. Open Kodi > Add-ons > Install from zip file
3. Select the Genesis addon zip
4. Configure your debrid service in Settings > Debrid Services

## Configuration
1. Go to Add-ons > Genesis > Settings
2. Authorize your preferred debrid service(s)
3. Authorize Trakt for watchlist sync (optional)
4. Enable/disable torrent scrapers as needed

## Requirements
- Kodi 17.0 or higher
- A debrid service subscription (Real-Debrid, Premiumize, AllDebrid, or TorBox)
