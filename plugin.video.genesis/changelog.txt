v1.5.8 (2026-05-09) - TORBOX PROVIDER REWRITE
=============================================
The TorBox debrid integration has been rewritten from
scratch to match the public TorBox API surface used by
the wider community. The old device-code flow is gone
because the official TorBox endpoint is just a long-lived
API key copied from the user's account page.

NEW:
- Manual API key authorization. The "Authorize TorBox"
  action now opens an input dialog. Paste the API key
  from https://torbox.app/settings - that's it.
- Smart magnet resolver:
    * file selection by extension whitelist
    * season/episode pattern matching for TV packs
    * sample / trailer / extras filtering
    * .m2ts safety check (rejects Blu-ray folder rips)
    * automatic torrent cleanup unless caching is on
- POST /torrents/checkcached with hash list and
  ?format=list - much faster than per-hash polling.
- Direct stream URLs via GET /torrents/requestdl
  (token + torrent_id + file_id), the canonical TorBox
  flow used by every other Kodi addon.
- Cloud browser now lists both Torrents and Usenet
  downloads side by side.
- Account dialog now shows plan name (Free, Essential,
  Pro, Standard), expiry date and total bytes downloaded.

FIX:
- Replaces the previous device-code flow that never
  produced a usable token because TorBox does not
  implement RFC 8628 device authorization.

SETTINGS:
- TorBox section simplified to a single "Set TorBox
  API Key" action. The API key field is hidden.
