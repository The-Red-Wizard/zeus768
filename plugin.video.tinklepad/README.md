# Tinklepad v1.0.1
**Author:** Zeus768  
**Theme:** Golden Eagle Premium  

Tinklepad is a high-end Kodi video addon built on a modernized Exodus architecture. It features a curated cinematic experience with a focus on premium aesthetics and deep content organization.

## 🦅 Key Features
- **Little Eagles:** A dedicated, age-filtered sanctuary for family and children's content.
- **Global Networks:** Direct access to the top 20 streaming and broadcast networks worldwide.
- **Cinematic Intro:** Custom startup video integration for a premium feel.
- **Golden Eagle UI:** Every menu, label, and dialog is themed with the signature Zeus768 Gold.
- **Unlimited Scroll:** Pagination logic integrated into all categories for endless browsing.
- **Custom Source Select:** A unique WindowXML dialog for selecting streams.

## 📁 Installation & Structure
1. Ensure all dependencies (`script.module.resolveurl`, `script.module.requests`) are installed.
2. Place your `startup.mp4` and all 34 custom icons in `resources/media/`.
3. Enter your **TMDB API Key** in the addon settings to enable metadata and search.
