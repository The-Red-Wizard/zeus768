# SALTS - Stream All The Sources
## Modernized for Kodi 21+ by zeus768

![SALTS](icon.png)

SALTS is a multi-source video addon that aggregates content from various torrent sites and supports debrid services for cached torrent playback.

## Features

### Torrent Site Scrapers
- **1337x** - One of the most popular torrent sites
- **YTS/YIFY** - High quality movie torrents
- **EZTV** - Dedicated TV show torrents
- **TorrentGalaxy** - General torrent site
- **ThePirateBay** - Classic torrent repository
- **LimeTorrents** - General torrents
- **Torrentz2** - Meta-search engine
- **RARBG** - Mirrors of the original site
- **Nyaa** - Anime torrents

### Indexer Aggregators
- **Jackett** - Connect to 400+ torrent indexers through one interface
- **Prowlarr** - Alternative to Jackett with similar functionality

### Debrid Services
- **Real-Debrid** - Cache checking and instant playback
- **Premiumize** - Premium torrent caching
- **AllDebrid** - Alternative debrid service

## Installation

1. Download the latest release zip file
2. In Kodi, go to Add-ons > Install from zip file
3. Navigate to the downloaded zip and select it
4. Configure your preferred scrapers and debrid services in Settings

## Configuration

### Debrid Services
1. Go to Settings > Debrid Services
2. Enable your preferred service
3. Follow the authorization prompts

### Jackett/Prowlarr
1. Install Jackett or Prowlarr on your system
2. Configure your torrent indexers
3. In SALTS Settings > Scrapers > Indexer Aggregators
4. Enter your server URL and API key

## Requirements

- Kodi 21 Omega or newer
- script.module.resolveurl
- script.module.requests
- script.module.beautifulsoup4
- script.module.six
